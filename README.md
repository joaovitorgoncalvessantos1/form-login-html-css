# 🔐 Formulário de Login Moderno

## 📖 Sobre o projeto
Este é um projeto de interface de login desenvolvido com foco em design moderno e prática de HTML e CSS.

A proposta foi criar um formulário visualmente agradável, utilizando efeitos de transparência (glassmorphism) e uma boa organização dos elementos na tela.

---

## ✨ Funcionalidades
- Campo de email
- Campo de senha
- Botão de login
- Opção "Lembrar-me"
- Link "Esqueci minha senha"

---

## 🎨 Layout
O projeto possui:
- Design moderno e minimalista
- Efeito glass (transparência)
- Fundo com imagem + overlay escuro
- Card centralizado

---

## 🛠️ Tecnologias utilizadas
- HTML5
- CSS3

---

## 🎯 Aprendizados
Durante esse projeto, pratiquei:
- Estruturação de formulários
- Estilização com CSS
- Centralização de elementos
- Criação de interfaces modernas
- Uso de transparência e efeitos visuais
